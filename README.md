# 🎬 StreamVerse

A modern Netflix-style movie streaming web application built with Next.js and the TMDB API.

StreamVerse delivers a cinematic browsing experience with dynamic categories, real-time search, and responsive UI inspired by modern OTT platforms.

---

## live demo
https://streamverse-blue.vercel.app

---

## 🚀 Features

- 🔍 Real-time movie search
- 🎞 Trending, Top Rated & Upcoming categories
- ▶️ Trailer playback integration
- 📱 Fully responsive design
- ⚡ Fast performance with Next.js
- 🎨 Clean, modern UI inspired by Netflix

---

## 🛠 Tech Stack

- **Frontend:** Next.js (App Router)
- **API Integration:** TMDB (The Movie Database)
- **Styling:** Tailwind CSS
- **State Management:** React Hooks
- **Deployment:** (Add after deploying – e.g., Vercel)

## ⚙️ Installation

```bash
git clone https://github.com/swathivhu/streamverse.git
cd streamverse
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env.local` file and add:

```
NEXT_PUBLIC_TMDB_API_KEY=your_tmdb_api_key
```
